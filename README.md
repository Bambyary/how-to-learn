# *Научиться учиться*
---

Проект "Научиться учиться" - это одностраничный сайт, созданный с помощью HTML и CSS, с использованием:
```
* Флекс-бокс вёрстки;
* Позиционирования элементов;
* Технологии БЭМ;
* API и iframe;
* Анимаций.
```
_В ближайшее время сайт будет адаптирован и под мобильные устройства._
